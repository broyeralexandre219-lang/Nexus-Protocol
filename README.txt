# 🎮 Nexus Protocol

> A cyberpunk Puzzle RPG inspired by the gameplay of Yo-kai Watch Puni Puni.

## 📖 About

Nexus Protocol is a puzzle/action game developed with **Godot 4.7**.

Create chains of Cores, trigger massive explosions and defeat corrupted digital entities called **Viruses**.
